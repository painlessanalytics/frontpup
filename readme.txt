== Description ==

FrontPup allows you to maximize the settings available between the CloudFront Origin and your WordPress site. 

Features include:

* Set no-cache headers for all pages (great for development or testing)
* Set public Cache-Control headers for CloudFront and Browsers
* Set private Cache-Control headers for Browsers only
* Set separate max-age and s-maxage values

== Frequently Asked Questions ==

= Do I need an AWS CloudFront to use this plugin? =

Yes, you need an AWS account with CloudFront setup for your website.
